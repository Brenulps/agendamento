# Fluxo de Gerenciamento de Especialidades

Este workflow descreve como gerenciar as áreas de atuação disponíveis no sistema.

## 1. Acessar a Página
1. No menu lateral, clique em **Especialidades**.

## 2. Adicionar Especialidade
1. Clique no botão **Nova Especialidade**.
2. Digite o nome da especialidade no campo indicado.
3. Clique em **Salvar**.
4. A lista será atualizada e um ícone representativo será atribuído automaticamente com base no nome.

## 3. Editar Especialidade
1. Clique no ícone de **lápis (Editar)** no card da especialidade.
2. Altere o nome conforme necessário.
3. Clique em **Salvar**.

## 4. Excluir Especialidade
1. Clique no ícone de **lixeira (Excluir)** no card.
2. Confirme a remoção no modal de confirmação.
