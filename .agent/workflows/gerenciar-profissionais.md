# Fluxo de Gerenciamento de Profissionais

Este workflow descreve como vincular usuários a especialidades para torná-los profissionais no sistema.

## 1. Acessar a Página
1. No menu lateral, clique em **Profissionais**.

## 2. Vincular Novo Profissional
1. Clique no botão **Novo Profissional**.
2. No modal:
    - **Usuário**: Selecione um usuário já cadastrado.
    - **Especialidade**: Selecione a especialidade que será atribuída a ele.
3. Clique em **Vincular Profissional**.
4. Verifique se o profissional aparece na listagem com seu respectivo card.

## 3. Remover Profissional
1. Localize o card do profissional.
2. Passe o mouse sobre o card e clique no ícone de **lixeira (Excluir)**.
3. Confirme a ação no modal de segurança.
4. O usuário não será excluído do sistema, apenas perderá o vínculo de profissional.
