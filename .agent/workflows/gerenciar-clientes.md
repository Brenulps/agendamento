# Fluxo de Gerenciamento de Clientes

Este workflow descreve como operar a página de clientes, incluindo a navegação, criação, edição e remoção de registros.

## 1. Acessar a Página
1. No menu lateral, clique em **Clientes**.
2. A página carregará automaticamente todos os registros existentes.

## 2. Adicionar Novo Cliente
1. Clique no botão **Novo Cliente** no canto superior direito.
2. Preencha o formulário no modal:
    - **Nome**: Nome completo do cliente.
    - **E-mail**: Endereço de e-mail válido.
    - **Telefone**: Número de contato (opcional).
3. Clique em **Cadastrar Cliente**.
4. Verifique a mensagem de sucesso no canto inferior da tela.

## 3. Editar Cliente Existente
1. Localize o card do cliente que deseja alterar.
2. Passe o mouse sobre o card para revelar as ações.
3. Clique no ícone de **lápis (Editar)**.
4. Altere as informações desejadas no formulário.
5. Clique em **Salvar Alterações**.
6. Verifique a atualização imediata dos dados no card.

## 4. Excluir Cliente
1. Localize o card do cliente que deseja remover.
2. Passe o mouse sobre o card e clique no ícone de **lixeira (Excluir)**.
3. Leia as informações no modal de confirmação.
4. Clique em **Excluir Permanentemente**.
5. Verifique se o card foi removido da listagem.
